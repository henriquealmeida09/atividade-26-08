Regra da Cascata

Determina qual estilo será aplicado quando houver conflitos, seguindo esta ordem:

  • Especificidade do seletor.
  • Ordem do código (último prevalece).

Regra da Especificidade

Define a prioridade de um seletor.Exemplo de pesos:

  • Inline style: 1000
  • ID: 0100
  • Classe/pseudo-classe: 0010
  • Tag: 0001

    • Quem tiver maior peso, prevalece no site.

Regra da combinação de seletores

A combinação de seletores no CSS é quando você usa dois ou mais seletores juntos para selecionar elementos de forma mais específica.

  • .primeiro h1: Qualquer h1 dentro de .primeiro
  • .primeira > h1: Apenas h1 filho direto de .primeira
  • .primeira h1 + h1: h1 logo após outro h1 dentro de .primeira

Regra das pseudo-classes

Uma pseudo-classe em CSS é uma palavra-chave que você adiciona a um seletor para estilizar elementos com base em seu estado ou posição.

  • :first-child: Primeiro filho do pai
  • :last-child: Último filho
  • :nth-child(n): N-ésimo filho
  • :nth-of-type(n): N-ésimo do mesmo tipo

Regra dos Displays

A propriedade display define como um elemento HTML será exibido na página, se ele ocupa uma linha inteira, se fica ao lado de outros, se se comporta como um container flexível, etc.

Display Inline

  • Largura:Ocupa apenas o conteúdo.
  • Margin/padding:Apenas horizontal
  • Pode conter elementos dentro dele

Display Block

  • Largura:Ocupa a largura inteira
  • Margin/padding:Todos os lados
  • Não pode conter elementos dentro dele

Elementos com display padrão

Display Inline:span, a, strong, em, b, i.

Display Block:div, p, h1 a h6, section, article, header, footer, main, ul, ol, li.

Margin, Padding e Border + Shorthand

Função:

  • Margin: Espaço externo ao elemento
  • Padding: Espaço interno, entre o conteúdo e a borda
  • Border: Linha ao redor do elemento

Shorthand é a ordem dos valores.

Direções Específicas:

  • margin-top
  • margin-rigth
  • margin-bottom
  • margin-left
  • padding-top

Utilização da propriedade box-sizing: border-box

A propriedade box-sizing: border-box facilita o controle do tamanho de um elemento, especialmente ao usar padding e border.
